Before you begin

BOLO Writer uses the OpenAI API to generate text. To use the program, you need your own OpenAI API key.

How to get an OpenAI API key

  1. Go directly to OpenAI's API keys page:
    https://platform.openai.com/api-keys
  2. Sign in to your OpenAI account.
  3. Click Create new secret key.
  4. Copy the key and store it safely. You will not be able to view it again later.
  5. Add billing in OpenAI and fund the account as needed.
  6. Paste your API key into BOLO Writer and start generating.
A rough rule of thumb is that one page of generated text costs about $0.01 in API usage, assuming roughly 300 words per page.

What BOLO Writer is for

BOLO Writer is designed to make the full workflow around AI-assisted book production simpler and more structured. It is especially well suited to nonfiction books, guidebooks, geographic books, historical projects, and other long-form factual writing.

BOLO Writer is built for structured long-form writing, not for fiction and not for short chat-style text generation.
