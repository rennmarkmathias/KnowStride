Before you begin

BOLO uses the OpenAI API to generate text. To use the program, you need your own OpenAI API key.

How to get an OpenAI API key

  1. Go directly to OpenAI's API keys page:
    https://platform.openai.com/api-keys
  2. Sign in to your OpenAI account.
  3. Click Create new secret key.
  4. Copy the key and store it safely. You will not be able to view it again later.
  5. Add billing in OpenAI and fund the account as needed.
  6. Paste your API key into BOLO and start generating.
A rough rule of thumb is that one page of generated text costs about $0.01 in API usage, assuming roughly 300 words per page.

What BOLO is for

BOLO is designed to make AI-assisted book production simpler, more structured and more predictable. It is especially well suited to nonfiction books, guidebooks, educational material, geographic books, historical projects and other long-form factual writing.

BOLO is optimized for structured nonfiction workflows. Fiction support exists in beta, but the product is not positioned as a chat-style writing tool.
