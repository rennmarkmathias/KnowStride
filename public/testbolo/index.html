Innan du börjar

BOLO använder OpenAI:s API för att generera text. För att använda programmet behöver du därför en egen API-nyckel från OpenAI.

Så skaffar du en OpenAI API-nyckel

  1. Gå direkt till OpenAI:s API-nyckelsida:
    https://platform.openai.com/api-keys
  2. Logga in eller skapa konto.
  3. OpenAI tillåter för närvarande endast inloggning via:
    • Google
    • Apple
    • Microsoft
    Det finns i nuläget inga andra registreringsalternativ med vanlig e-post och lösenord.
  4. Klicka på Create new secret key.
  5. Kopiera och spara nyckeln på ett säkert ställe. Du kan inte se den igen senare.
  6. Gå till Billing och lägg till ett kort, ladda $5-20.
  7. Klart – nu kan du använda OpenAI:s API i BOLO.
En grov tumregel är att en sida text kostar ungefär $0.01 i API-kostnad vid cirka 300 ord per sida.

Vad BOLO gör

BOLO är byggt för att göra hela arbetsflödet kring AI-assisterad bokproduktion enklare och mer strukturerat. Det passar särskilt bra för fackböcker, guideböcker, geografiska böcker, historiska projekt och andra typer av längre sakprosa.

Programmet är gjort för skrivande i längre format – inte för skönlitteratur och inte för korta chattliknande texter.
